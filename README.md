# Career Path Vault

Этот каталог подготовлен как Obsidian-совместимый вольт для двух учебных треков:

- [[cpp-junior/README|C++ Junior Track]]
- [[rust-junior/README|Rust Junior Track]]

Цель обоих треков: довести знания до уровня, достаточного для собеседований и первой работы на позиции junior/stage trainee.

- `market-skills/` - Git, CMake, Linux CLI, debugging, TCP/IP

## Как работать

1. Открой нужный трек.
2. Иди по модулям сверху вниз.
3. После каждого модуля выполняй практику в отдельном каталоге с кодом рядом с этим vault или в любом удобном репозитории.
4. Возвращайся сюда и отмечай прогресс в чеклистах.
5. Для проверки решений открывай файл восстановления контекста и продолжай работу с Codex.

## Базовый режим

- C++: старт с уровня "знаю синтаксис, практики мало".
- Rust: старт с нуля.
- Темп: 5-6 занятий в неделю по 60-120 минут.
- Ожидаемый горизонт до уверенного junior уровня:
  - C++: 12-16 недель
  - Rust: 14-20 недель

## Как пользоваться с телефона

- Храни каталог как обычный git-репозиторий или как часть репозитория.
- Не добавляй тяжелые бинарные файлы.
- Все материалы в Markdown, ссылки относительные, поэтому Obsidian Mobile сможет открыть их без доработок.
