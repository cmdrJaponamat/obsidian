## Почему эти курсы вынесены отдельно

По вакансиям hh.ru на 2025-2026 регулярно встречаются:

- Git
- Linux
- CMake или другие системы сборки
- gdb/отладка
- понимание сетей и TCP/IP

Это не "приятные дополнения", а реальные фильтры найма, особенно для C++-ролей под Linux. Для Rust эти навыки тоже сильно повышают шансы, даже если основной стек вакансии смещен в backend и async.
