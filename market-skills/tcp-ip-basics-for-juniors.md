# Курс: TCP/IP база для junior

## Почему это нужно рынку

Для C++ вакансий под Linux и системную разработку сеть всплывает очень часто. Для Rust backend ролей тоже регулярно нужны HTTP, TCP/UDP и базовое понимание сетевого взаимодействия.

## Цель

Собрать понятную практическую сетевую базу без лишней телеком-теории.

## Теория

### Что понимать

- IP-адрес и порт
- клиент и сервер
- TCP vs UDP
- что такое сокет
- localhost
- DNS на базовом уровне
- HTTP как прикладной протокол поверх TCP

### Что полезно знать для собеседования

- почему TCP надежный, а UDP нет
- что такое connection
- почему таймауты и retries важны
- зачем логировать сетевые ошибки

## Практика

1. Напиши самый простой TCP echo client/server:
   - на C++ или Rust
   - локально на `127.0.0.1`
2. Посмотри открытые порты на системе.
3. Сделай HTTP-клиент:
   - запрос
   - статус-код
   - разбор ответа
4. Смоделируй сетевую ошибку и обработай ее.

## Что нужно уметь после курса

- объяснить базовую сетевую модель
- не путать TCP и HTTP
- читать типовые сетевые ошибки
- написать и запустить простейшее клиент-серверное демо
