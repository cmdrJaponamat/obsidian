# Rust Junior Track

Точка старта: полный ноль в Rust.

Цель: выйти на уровень, где ты можешь писать нормальные CLI/утилиты/небольшие сервисы, понимать ownership и проходить junior или intern интервью по Rust.

## Что будет в результате

- понимание ownership без заучивания "магии"
- умение писать CLI-приложения
- базовое владение `serde`, `tokio`, обработкой ошибок
- привычка пользоваться `cargo fmt`, `clippy`, `test`
- проект, который можно показывать
