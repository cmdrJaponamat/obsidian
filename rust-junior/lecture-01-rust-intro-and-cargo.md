# Лекция 1. Введение в Rust, Cargo и базовый синтаксис

## Зачем вообще Rust

Rust востребован там, где нужны:

- безопасность памяти без garbage collector
- системное программирование
- backend и сетевые сервисы
- CLI-инструменты
- надежность и предсказуемость

Для тебя сейчас важен не весь Rust, а вход в язык без паники. Первая цель - начать свободно создавать проекты, запускать их и понимать базовый синтаксис.

## Что делает Rust необычным

Если очень коротко:

- язык строго типизирован
- компилятор очень активно проверяет корректность
- память и владение контролируются через ownership model

Из-за этого вход кажется жестче, чем в некоторых других языках, но потом многие ошибки отсекаются до запуска программы.

## Cargo

Cargo - это стандартный инструмент Rust для:

- создания проекта
- сборки
- запуска
- тестов
- управления зависимостями

Это огромное преимущество экосистемы: базовый workflow стандартизирован.

### Основные команды

- `cargo new app_name`
- `cargo run`
- `cargo check`
- `cargo test`
- `cargo fmt`

Особенно полезна команда `cargo check`: она проверяет код быстрее, чем полная сборка.

## Минимальная программа

```rust
fn main() {
    println!("Hello, Rust!");
}
```

Здесь:

- `fn` объявляет функцию
- `main` - точка входа
- `println!` - макрос вывода

## Переменные и изменяемость

По умолчанию переменные неизменяемые.

```rust
let x = 10;
```

Если нужно менять значение:

```rust
let mut counter = 0;
counter += 1;
```

Это хороший default: ты сразу видишь, какие данные реально меняются.

## Базовые типы

На старте полезно знать:

- `i32`, `i64`, `usize`
- `f64`
- `bool`
- `char`
- `String`
- `&str`

Пример:

```rust
let age: i32 = 21;
let price: f64 = 19.99;
let is_ready: bool = true;
let title: &str = "Rust";
```

## `String` и `&str`

Это одна из первых важных тем.

Упрощенно:

- `&str` - строковый срез, ссылка на строковые данные
- `String` - владеющая, изменяемая строка

Примеры:

```rust
let name: &str = "Ivan";
let mut owned = String::from("hello");
owned.push('!');
```

На первом этапе достаточно помнить:

- литералы обычно `&str`
- если нужно владеть строкой и менять ее, часто нужен `String`

## Условные конструкции и циклы

### `if`

```rust
if age >= 18 {
    println!("adult");
} else {
    println!("minor");
}
```

### `loop`

```rust
let mut count = 0;
loop {
    count += 1;
    if count == 3 {
        break;
    }
}
```

### `while`

```rust
let mut x = 3;
while x > 0 {
    println!("{x}");
    x -= 1;
}
```

### `for`

```rust
for value in [1, 2, 3] {
    println!("{value}");
}
```

## Функции

```rust
fn square(value: i32) -> i32 {
    value * value
}
```

Обрати внимание:

- типы аргументов указываются явно
- тип возвращаемого значения идет после `->`
- последнее выражение можно вернуть без `return`

## `match`

Это один из центральных инструментов Rust.

```rust
let number = 2;

match number {
    1 => println!("one"),
    2 => println!("two"),
    _ => println!("other"),
}
```

`match` важен, потому что помогает явно описывать варианты состояний.

## Что полезно делать с первого дня

- запускать `cargo check` очень часто
- сразу форматировать код через `cargo fmt`
- не пытаться запомнить весь язык заранее
- читать текст ошибок компилятора внимательно

## Типичные ошибки новичка

- путать `String` и `&str`
- бояться сообщений компилятора вместо того, чтобы разбирать их
- писать все в одном длинном `main`
- забывать про `mut`
