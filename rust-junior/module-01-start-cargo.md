# Модуль 1. Старт, Cargo, синтаксис

## Цель

Перестать бояться нового синтаксиса и начать свободно запускать простые программы.

## Теория

### Что выучить в первую очередь

- `let`, `mut`
- базовые типы
- `if`, `match`, `loop`, `while`, `for`
- функции
- строки и срезы
- `Vec`
- структура проекта Cargo

### Инструменты

- `cargo new`
- `cargo run`
- `cargo check`
- `cargo test`
- `cargo fmt`

## Практика

1. Создай проект `calc_rs`.
2. Реализуй калькулятор для `+ - * /`.
3. Добавь чтение аргументов командной строки.
4. Добавь проверку ошибок на неправильный ввод.

## Мини-памятка

- `cargo check` запускай чаще, чем `cargo build`
- форматируй код сразу
- не пытайся выучить весь синтаксис за один день

- Понимаю ли я разницу между `String` и `&str` на базовом уровне?
- Могу ли я запустить и собрать проект без подсказки?
