# Модуль 5. Тесты, CLI, файлы

## Цель

Научиться делать законченные утилиты, а не только учебные функции.

## Теория

- unit tests
- integration tests
- `assert_eq!`
- `#[derive(Debug, PartialEq)]`
- чтение и запись файлов
- базовый CLI через `std::env` или `clap`

## Практика

1. Сделай CLI-программу заметок с сохранением в JSON.
2. Подключи `serde` и `serde_json`.
3. Реализуй команды:
   - add
   - list
   - done
   - remove
4. Напиши:
   - минимум 5 unit tests
   - минимум 2 integration tests

## Проверочный список

- ошибки не теряются
- формат хранения понятен
- проект можно запустить по README
- тесты не хрупкие
