# Модуль 7. Реальные практики

## Цель

Собрать рабочие привычки, без которых junior Rust быстро упирается в хаос.

## Теория

- `cargo fmt`
- `cargo clippy`
- `thiserror` и `anyhow`
- логирование
- конфигурация через env
- `README`, примеры запуска, ограничения проекта

## Практика

1. Возьми один из предыдущих проектов.
2. Добавь:
   - нормальные типы ошибок
   - логирование
   - README
   - примеры использования
3. Прогони `cargo clippy -- -D warnings`.
4. Убери лишние `clone()` и неясные имена.

## Устная часть

Подготовься кратко объяснять:

- ownership
- borrowing
- `Option` vs `Result`
- зачем нужен `trait`
- когда использовать `Vec`, а когда `HashMap`
