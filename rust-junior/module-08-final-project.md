# Модуль 8. Финальный проект

## Цель

Сделать проект, который подтверждает, что ты не просто читал про Rust, а умеешь на нем работать.

## Варианты проекта

1. `todo-sync-cli`
2. `expense-tracker-cli`
3. `http-data-fetcher`
4. `log-parser`

## Обязательные требования

- Cargo-проект с нормальной структурой
- `Result`-ориентированная обработка ошибок
- тесты
- README
- сериализация или работа с файлами
- хотя бы одна внешняя библиотека по делу

## Желательные требования

- async через `tokio`
- конфиг
- логирование
- интеграционные тесты

## Что показать мне на проверку

- `Cargo.toml`
- структура модулей
- ключевые части `src/`
- тесты
- README

## Финальная самопроверка

- Могу ли я объяснить, где ownership, а где borrow?
- Нет ли в коде бессмысленных `clone()`?
- Могу ли я рассказать, почему выбрал эти типы и crates?
